# ✅ Web-40 解題說明 — Speech

## 題目描述
最終題刻意讓直接下載失效，挑戰者需改走 DevTools Network 取得真正檔案，再進行資料轉換與格式判斷，完成最後旗標還原。

## 難度
★★★★☆（4星）

## 種類
WEB, MISC

## 建議工具
- 瀏覽器 DevTools（Network）
- CyberChef（From Base64）
- 音訊播放器 / ffprobe

## 解題步驟
1. 開啟 `http://localhost:8040`，按 `F12`。
2. 在 `Network` 觀察並下載 `speech.dat`。
3. 讀取內容後，確認為可解碼字串（如 Base64）。
4. 用 CyberChef 轉回原始資料，必要時再另存為可播放格式。
5. 依題目提示還原最終旗標。

## 驗證與常見卡點
- 驗證方式：輸出需是正確 `flag{...}`。
- 常見卡點：只點頁面按鈕，不從 Network 取檔會卡關。
- 常見卡點：副檔名可能是誤導，需判斷實際資料型態。

## 學習重點
- 前端互動可能是誘餌，Network 才是真實線索來源。
- 綜合題重點在流程判斷而非單一工具。
- 檔案格式辨識與轉換是實戰核心能力。
