# ✅ Web-10 解題說明 — 多步轉換壓軸題

## 題目描述
最後一題給你一段多層編碼字串。請依線索判斷正確解碼順序，在 CyberChef 逐步轉換後取得最終旗標。

## 解題步驟
1. 在頁面註解、標頭或 JS 找到主字串與順序提示。
2. 將主字串貼入 CyberChef。
3. 依提示順序套用操作（例：`From Base64` → `URL Decode` → `From Hex`）。
4. 若結果仍非 `flag{...}`，檢查是否順序顛倒。
5. 成功還原旗標後提交。

## 學習重點
- 多層編碼解題核心是「辨識格式 + 正確順序」。
- 記錄每步輸出可快速定位錯誤。
