# ✅ Web-38 解題說明 — LSB 影像隱寫術

## 題目描述
此題把資訊藏在圖片像素的最低有效位元（LSB）。圖片外觀看起來正常，但每個像素 bit 被微調後可拼出可讀內容。

## 難度
★★★☆☆（3星）

## 種類
WEB, MEDIA, STENO

## 建議工具
- 瀏覽器 DevTools（Network）
- LSB 工具（futureboy.us/stegano / zsteg）
- Python（Pillow，自行抽位元）

## 解題步驟
1. 開啟 `http://localhost:8038`。
2. 下載題目提供的圖片檔。
3. 使用 LSB 工具抽取隱藏位元資料。
4. 若輸出是編碼字串，做 Base64/Hex 等後續轉換。
5. 還原成最終旗標。

## 驗證與常見卡點
- 驗證方式：輸出需符合 `flag{...}`。
- 常見卡點：通道順序（RGB/BGR）或位元順序不同會導致亂碼。
- 常見卡點：檔名副檔名與實際格式可能不一致，先確認檔頭。

## 學習重點
- LSB 隱寫利用「人眼不易察覺微小像素差異」。
- 圖像隱寫常需嘗試多種抽取策略。
- 工具與手寫腳本都值得掌握。
